# Security Log Analysis with Python

## Overview

This project analyzes Linux authentication logs and identifies suspicious login activity.

## Features

- Detect failed SSH login attempts
- Extract attacker IP addresses
- Identify targeted usernames
- Generate CSV security reports
- Basic SOC-style log analysis

## Technologies

- Python
- Regular Expressions
- CSV Reporting
- Linux Authentication Logs

Top Attacking IPs:
